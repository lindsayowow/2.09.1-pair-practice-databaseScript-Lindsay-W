# 2.09.1-pair-practice-databaseScript-Lindsay-W

Database Design  
Pair Practice: Using your Database Script 
Instructions 
For this activity, you will need to make sure you and your partner have both 
completed the Practice: Generating a Database Script activity. 
Using both your script and your partner’s, create and adapt a database for one of 
the following scenarios: 
1. You're developing a database for "Urban Eats," a restaurant chain that 
operates multiple locations across the city. Each restaurant has its own menu 
(though some items overlap), and they need to track customer orders across 
all locations for analytics, inventory management, and customer loyalty 
programs. 
2. You’re developing a library system that needs to expand to serve multiple 
departments in a university, tracking both library resources and course 
materials. 
3. You're building a database for "Premier Events Co.," an event planning 
company that manages various types of events across multiple venues in the 
region. They handle everything from corporate conferences and weddings to 
concerts and trade shows. The company needs to track venue bookings, 
manage ticket sales, and coordinate with different venue partners. 
Setup 
For this activity, you will need to use the repositories you and your partner 
completed for the Practice: Generating a Database Script activity. 
1. Create a new branch. 
2. Name it: pair-practice-databaseScript-[Your First Name]-[Your Last Initial] 
3. Make sure your partner is added as a collaborator on the repo. 
4. Clone the repository to your local machine. 
5. Start working on the code.
